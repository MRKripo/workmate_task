Анализатор csv файлов на рейтинг брендов

Команда для запуска:
python main.py --files [ЗАМЕНИТЬ ЭТИ КВАДРАТНЫЕ СКОБКИ НА ФАЙЛЫ/ПУТИ К ФАЙЛАМ. СКОБКИ ТОЖЕ УБРАТЬ] --report average-rating

В корневой папке проекта есть png с примерами запуска.
Если нужно запускать файлы без написания пути (просто сразу имя+тип файла). То они должны быть в одной и той-же папке где и сам скрипт питона. 
Аргумент report, не обязательный и скрипт может работать без него (будет красиво показывать содержимое всех файлов, которые ему дали.)
Также в этот самый report вшито приведение ввода в заглавные буквы. Так что даже если ввести "--report aVERAGE-RATING", он будет работать. 
Возможно отдельно надо будет устанавливать библиотеку tabulate.
